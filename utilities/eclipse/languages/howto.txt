Como configurar el paquete de idioma español para eclipse

ir a:
Windows -> 
 Preferences -> 
  General -> 
   Editors -> 
    Text Editors -> 
     Spelling	
o:
Windows -> 
 Preferences -> 
  Buscar: spelling

1. Poner en parte donde dice User defined dictionary, poner la 
   ubicacion del archivo ES.utf8.txt
2. Poner en el encoding UTF-8 , si no es la codificación por defecto ir a Other UTF-8
3. Reiniciar eclipse (File -> Restart)

OBS: Si por alguna razón no se pudiese usar la codificación UTF-8 se provee un fichero 
alternativo ES.iso8859-1.txt con el cuál configurar el diccionario
